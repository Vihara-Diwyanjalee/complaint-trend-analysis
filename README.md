# customer_complaint_analysis_dashboard

This project analyzes customer complaints using Power BI and Excel VBA to identify patterns in defects, sections, and monthly trends. The dashboard provides quick insights for improving product quality and supporting data-driven decision-making in the packing/manufacturing process.

## Steps

1. Data Collection
   - Complaint records
   - Defect categories
   - Section-wise logs
   - Monthly/periodic complaint data
2. Data Preprocessing
   - Cleaning and formatting raw complaint entries
   - Standardizing defect categories
   - Handling missing values
   - Preparing data models for Power BI
3. Dashboard Development
   - Section-wise complaint distribution
   - Defect type comparisonNaive Bayes
   - Monthly trend analysis
   - Recurring issue identification
4. Automation
   - Used Excel VBA to automate data updates
   - Linked processed data to Power BI for dynamic refresh
5. Insights
   - Helps identify high-risk sections
   - Highlights frequent defect patterns
   - Shows complaint spikes across months
   - Supports root-cause analysis and improvement decisions

## Tools & Technologies
- Power BI
- Excel (VBA)
- Data Cleaning & Modeling
